# Development Journals

Notes, intermediate results, logs, etc. during the development are kept here.

Jupyter notebooks are preferred because it encapsulates text, plots, code together in a single file.  Use <https://github.com/googlecolab/open_in_colab> to ease the launch of notebook in colab.  Markdown is acceptable but don't keep plots/images it uses under this directory.  Reference results elsewhere where they are properly categorized.

Keep each file to a reasonable size, for instance one file per week. 
 Follow the naming convention below.

## Naming convention

```
name_YYYYMM_a.ipynb
ymei_202505_a.ipynb
```
